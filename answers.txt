Q1. When I type google.com and press enter, my browser asks the DNS for Google's IP address.
    The browser then sends a request to Google's server, and the server sends the webpage back to my browser for me to see it.

 Q3. DNS stands for Domain Name System. We need it because it changes

 Q4. A firewall protects a network by blocking harmful or unauthorized traffic.
     A load balancer shares incoming traffic among several servers so that one server does not become overloaded.
